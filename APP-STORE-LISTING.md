## CAPACITOR.JS — WRAPPING NEXT.JS FOR APP STORE & PLAY STORE

### Overview
KOOL Events is a Next.js web app. To publish to the App Store and Google Play, it needs to be wrapped as a native shell. **Capacitor.js** is the recommended solution — it embeds your web app in a native iOS/Android container and gives you access to native APIs (camera, notifications, etc.) when needed.

**Alternative:** If the team prefers a fully managed pipeline, **Expo + EAS Build** with a web-view approach is also viable, but Capacitor integrates more cleanly with existing Next.js projects.

---

### Recommended: Capacitor.js

**What it does:**
- Wraps the Next.js app in a WKWebView (iOS) and WebView (Android)
- Gives native shell → App Store & Play Store can accept it
- Allows progressive enhancement with native plugins (push notifications, biometric auth, etc.)

---

### Setup Steps

#### 1. Export Next.js as Static Site (or use a URL)
```bash
# Option A: Static export (for offline-capable app)
# In next.config.ts:
# output: 'export'
# Then: npm run build → generates /out folder

# Option B: Load koolevents.app URL directly in WebView
# Simpler but requires internet; some reviewers may reject pure web views
```

**Recommendation:** Use static export OR a hybrid approach where the app loads the production URL but falls back gracefully.

#### 2. Install Capacitor
```bash
cd /Users/kool/.openclaw/workspace/kool-app
npm install @capacitor/core @capacitor/cli
npx cap init "KOOL Events" "app.koolevents.ios" --web-dir=out
```

#### 3. Add iOS & Android Platforms
```bash
npm install @capacitor/ios @capacitor/android
npx cap add ios
npx cap add android
```

#### 4. Build & Sync
```bash
npm run build       # generates /out (if using static export)
npx cap sync        # copies web assets into native projects
```

#### 5. Open in Xcode / Android Studio
```bash
npx cap open ios      # opens Xcode
npx cap open android  # opens Android Studio
```

#### 6. Configure App in Xcode
- Set Bundle ID: `app.koolevents.ios`
- Set version & build number
- Add App Icon set (1024×1024 source)
- Add Launch Screen
- Configure capabilities (Push Notifications, Sign in with Apple if needed)
- Archive → Upload to App Store Connect

#### 7. Configure App in Android Studio
- Set Application ID: `app.koolevents`
- Set version name & code
- Add adaptive icon
- Generate signed APK/AAB
- Upload to Google Play Console
